🚀 Sales Analysis with SQL & Python

A data analytics project focused on extracting insights from sales data using SQL and Python.

📌 Project Overview

This project analyzes sales performance by building a PostgreSQL database, running SQL queries, and visualizing results in Python. It demonstrates end-to-end data analysis—from data storage to insights.

🛠️ Tech Stack
SQL (PostgreSQL) – data querying & aggregation
Python – data processing
Pandas – data manipulation
Matplotlib – data visualization
SQLAlchemy – database connection

📂 Dataset
Simulated sales dataset
Fields include:
product_name
quantity
unit_price
sale_date

📊 Key Analysis Performed
Total sales per day
Revenue per product
Units sold per product
Revenue contribution (%) per product

📈 Key Insights
💰 Laptop generates the highest revenue
🖱️ Mouse has the highest sales volume
📅 Sales trends vary by day, showing peak period

Connecting PostgreSQL with Python
Data visualization best practices
Turning raw data into business insights
