# 🚀 Sales Analysis with SQL & Python

A data analytics project focused on extracting actionable insights from sales data using SQL and Python.

---

## 📌 Project Overview
This project analyzes sales performance by designing a PostgreSQL database, executing SQL queries, and visualizing results using Python. It showcases an end-to-end data analytics workflow—from data storage and transformation to insight generation.

---

## 🛠️ Tech Stack
- **SQL (PostgreSQL)** – data querying and aggregation  
- **Python** – data processing and analysis  
- **Pandas** – data manipulation  
- **Matplotlib** – data visualization  
- **SQLAlchemy** – database connection  

---

## 📂 Dataset
- Simulated sales dataset  

**Fields include:**
- `product_name`  
- `quantity`  
- `unit_price`  
- `sale_date`  

---

## 📊 Key Analysis Performed
- Daily total sales analysis  
- Revenue generated per product  
- Units sold per product  
- Revenue contribution (%) by product  

---

## 📈 Key Insights
- 💰 **Laptop** generates the highest revenue  
- 🖱️ **Mouse** has the highest sales volume  
- 📅 Sales trends fluctuate over time, highlighting peak sales periods  

---
